# Roll-Over Proof Car

This repository contains the conference paper and project details for a roll-over prevention system developed using MPU6050 sensor, Arduino, and servo motors.

## Overview

The project aims to improve vehicle stability and reduce roll-over risks by detecting tilt angles using an MPU6050 inertial sensor and activating servo-actuated stabilization mechanisms.

## Technologies Used

- Arduino
- MPU6050 Sensor
- Servo Motors
- Embedded Systems
- Tilt Angle Detection
